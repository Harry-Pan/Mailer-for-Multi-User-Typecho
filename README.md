用于 Typecho 博客程序具的评论邮件通知插件，仅支持 1.2.0 及以上版本

# 特性

高效：使用异步发信，不影响评论速度

简洁：后台简洁、代码简洁、核心代码不足300行

方便：保存插件配置自动发送测试邮件，自动记录发信错误日志可随时检查，正常发信无日志

# 说明

`宝塔/CDN防火墙`可能会拦截Typecho异步请求，也有可能是其他原因

解决方法（任选其一）：

1. 手动关闭异步模式
2. 将本机IP设置白名单
3. 防火墙关闭From-data协议
